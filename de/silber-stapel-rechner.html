So verwenden Sie den Batch-Rechner

Mit dem Chargenrechner können Sie mehrere Silbergegenstände hinzufügen und ihren Gesamtwert berechnen – ideal, wenn Sie den Verkauf einer Sammlung an einen Händler vorbereiten.

  1. Add items — Klicken Sie für jedes Stück auf „+ Weiteres Element hinzufügen“.
  2. Enter details — Beschreibung, Gewicht, Einheit und Reinheit für jeden
  3. See total — Die Gesamtsumme wird automatisch aktualisiert
  4. Print or copy — Bringen Sie die Quittung zu Ihrem Händler oder bewahren Sie sie zum Nachschlagen auf
