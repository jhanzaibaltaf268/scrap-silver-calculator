Como usar a calculadora de lote

A Calculadora de Lote permite adicionar vários itens de prata e calcular seu valor combinado – ideal quando você está se preparando para vender uma coleção a um revendedor.

  1. Add items — Clique em "+ Adicionar outro item" para cada peça
  2. Enter details — Descrição, peso, unidade e pureza de cada
  3. See total — O total geral é atualizado automaticamente
  4. Print or copy — Leve o recibo ao seu revendedor ou guarde-o para referência
💡 Pro Tip

Agrupe itens por pureza (todas as 925 peças juntas, todas as 800 peças juntas) para facilitar a negociação com os revendedores. Use o botão "Salvar no navegador" para manter seu inventário entre as visitas.
