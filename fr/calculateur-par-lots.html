Comment utiliser le calculateur de lots

Le calculateur de lots vous permet d'ajouter plusieurs objets en argent et de calculer leur valeur combinée – idéal lorsque vous vous préparez à vendre une collection à un revendeur.

  1. Add items — Cliquez sur "+ Ajouter un autre élément" pour chaque pièce
  2. Enter details — Description, poids, unité et pureté pour chacun
  3. See total — Le total général se met à jour automatiquement
  4. Print or copy — Apportez le reçu à votre revendeur ou conservez-le pour référence
💡 Pro Tip

Regroupez les articles par pureté (les 925 pièces ensemble, les 800 pièces ensemble) pour faciliter les négociations avec les revendeurs. Utilisez le bouton « Enregistrer dans le navigateur » pour conserver votre inventaire entre les visites.
